# proyecto-aplicacionPLN
Proyecto de PLN que analiza datos de Reddit extraídos con PRAW. Incluye clasificación de subreddits, cálculo de similitud de hilos, análisis de sentimientos y resúmenes abstractivos. Además, detecta contenido inapropiado usando SLMs (Gemma, Llama) mediante técnicas Zero-shot, Few-shot y Chain-of-Thought.
